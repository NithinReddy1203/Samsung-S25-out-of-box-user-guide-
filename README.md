# Galaxy S25 — First Power-On Guide

An illustrated, step-by-step guide to setting up a Samsung Galaxy S25 out of the box — from unboxing to home screen, including Wi-Fi, account sign-in, email setup, and everyday basics.

## Contents

| File | Description |
|---|---|
| `Galaxy-S25-First-Power-On-Guide.docx` | Printable/editable Word version with illustrations |
| `galaxy-s25-setup-guide.html` | Self-contained web version — open directly in a browser |

## What's covered

- What's in the box
- Inserting a SIM card / eSIM
- Powering on and language selection
- Connecting to Wi-Fi
- Transferring data with Smart Switch
- Signing in to Google and Samsung accounts
- Setting up email
- Screen lock and biometrics
- Reviewing privacy and terms
- Installing the latest software update
- Basic gestures, Quick settings, and everyday shortcuts

## Usage

- **Word:** Open `Galaxy-S25-First-Power-On-Guide.docx` in Microsoft Word, Google Docs, or LibreOffice. Edit freely — all illustrations are original line-art graphics, not screenshots, so they're safe to reuse and modify.

## Notes

Steps reflect the standard One UI setup wizard on the Galaxy S25. Exact wording and screen order may vary slightly by carrier and region.
